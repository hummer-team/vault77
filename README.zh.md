# Vaultmind

Vaultmind 是一个面向业务与数据团队的 **智能分析工作台**。  
它连接结构化数据（如 Excel / CSV、数仓），结合「角色 / 人设」驱动的智能 Agent，通过自然语言完成从提问、建模到结论说明的整条分析链路。

---

## 业务价值

- **面向业务用户**
  - 直接用自然语言提问，例如：「对比 Q4 和 Q3 各地区收入，并标出异常区域」。
  - 自动给出：
    - 清晰可读的结论说明
    - 对应的表格 / 图表
    - 可复现的分析步骤

- **面向数据团队**
  - 将指标口径、业务规则、分析套路沉淀为 **Persona / Agent**：
    - 统一指标和语义
    - 可审计、可追踪
    - 降低“帮忙出个报表”的重复劳动

- **面向组织**
  - 减少临时报表需求
  - 规范化指标与业务语言
  - 让决策围绕可信的数据分析快速发生

---

## 核心功能特性

### 1. Persona / Agent 体系

- 支持多种 **角色人设**（如「业务分析师」「财务经理」「增长 PM」）。
- 每个 Persona 包含：
  - 专业领域与术语习惯
  - 典型分析套路与框架
  - 风险与安全策略（可访问的数据、可用的工具等）
- 在前端以 **Persona 徽章** 形式呈现，当前人设将影响：
  - 提示词建议
  - 回答风格和详细程度
  - 工具调用策略和安全边界。

### 2. 自然语言分析

- 支持中英文自然语言提问，系统会：
  - 识别意图、约束条件
  - 规划多步分析任务
  - 将请求拆解为底层查询 / 转换 / 可视化操作
  - 最终给出带有推理过程的答案

- 典型场景：
  - 探索式分析（「最近一个月发生了什么变化？」）
  - 假设验证（「涨价是否显著影响了流失率？」）
  - 对现有报表的解释与问答。

### 3. 文件上传与数据加载

- 在 Chat 面板中直接上传 **Excel / CSV** 文件：
  - 支持多 sheet 的工作簿
  - 支持较大文件（约 1 GB，视后端配置而定）
- 自动按文件分组展示附件：
  - 单个文件中多个 sheet 的加载情况
  - 状态：上传中 / 成功 / 失败
  - 在标签和气泡提示中展示 sheet 名称、错误信息等。

### 4. 引导式提示与问题建议

- 根据：
  - 当前 Persona
  - 当前数据上下文
  - 历史会话
  提供 **提示建议 (Suggestions)**。

- 用户可一键点击标签快速开始常见分析。
- 输入区右侧支持 **Persona 提示文案**，引导用户提出更适合当前角色的问题。

### 5. Agent 执行引擎（概念说明）

> 本节为对 `agent.md` 设计的简要概述。

- **多工具 Agent 架构**：
  - SQL / DataFrame 查询工具
  - 数据清洗 / 转换工具
  - 图表 / 可视化工具
  - 说明 / 文档生成工具

- **规划与自我迭代**：
  - 先生成分析计划
  - 逐步执行工具调用
  - 检查中间结果
  - 必要时修正计划，直到给出满意的答案

- **可观测性**：
  - 每次会话回合都有结构化 Trace
  - 记录所有工具
